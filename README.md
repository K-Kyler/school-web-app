# Run Project Step-by-Step:
<br />1/ Clone project: 
```
git clone https://github.com/K-Kyler/school-web-app.git
```
<br />2/ Import node_modules:
```
cd school-web-app
npm install
```
<br />3/ Setup database by MongoDB with name "school-web-app"

<br />4/ Run:
```
npm start
```
